Pause

Waits the specified amount of time before continuing.

Pause (F5)

Parameters

Add Pause
The amount of time to pause (choose format in the options below). This field accepts Variables & Expressions.
Miliseconds / Seconds / Minutes
Sets format of time (seconds and minutes will be converted to miliseconds when the command is added).
Random delay
Sets the command use a random value between Minimum and Maximum instead of a predefined value.
Disable random delays for this command
If the global option for *Random delays" is active the command won't be affected.
