Download Files

Downloads one or more files from the Internet. All fields accept variables.

Download Files (Shift+F11)

Parameters

Destination folder
The name of the destination folder, which is assumed to be in %A_WorkingDir% if an absolute path isn't specified.
Download links
A list of download links to be downloaded. Separate links by semicolon, comma or line.

Remarks

This function is not guaranteed to work with any Url.

To bypass error messages go to Options > Settings > Playback and check the "Do not display errors" option.
